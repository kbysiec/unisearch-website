# Dodaj obrazy przed uruchomieniem

Wymagane pliki:

1. `public/assets/logo.png` — logo aplikacji
2. `public/assets/app-screenshot.png` — screenshot aplikacji
3. `public/og-image.png` — obraz OG (1200x630)
4. favicony (`public/favicon.ico`, `public/apple-touch-icon.png`, `public/android-chrome-*.png`)

Po podmianie:

```bash
npm run dev
```
